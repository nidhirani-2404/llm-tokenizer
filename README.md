# 🤖 LLM Tokenizer

A simple Python project to understand how **GPT models tokenize text** using the tiktoken library.

## ✨ Features

- **Encode** text into tokens
- **Decode** tokens back to text
- **Count** total tokens

## 📦 Installation

```bash
pip install -r requirements.txt
```


## 🚀 Run

```bash
python tokenizer.py
python detokenizer.py

```
